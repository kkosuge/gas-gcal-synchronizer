# gas-gcal-synchronizer
複数アカウントでGoogleカレンダーの予定を同期するGoogle Apps Script


## 使い方
### Google Apps Script へ登録する

```
yarn global add clasp
clasp create --rootDir ./src
yarn
clasp push
```
